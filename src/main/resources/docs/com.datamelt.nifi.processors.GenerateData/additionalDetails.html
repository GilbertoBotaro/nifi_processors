Processor to generate mass data based on word lists, regular expressions or purely random.

When the datagenerator generates random rows of data, it will pick a random word or expression from the category file. So you can create mutiple category files containing lists of words which the datagenerator will use. If you define a regular expression pattern like, then the datagenerator will create data that fits to the given pattern. Or one can define a field type of "random", where data will be generated purely random.

The structure of the data to generate is defined in an external Xml file (rowlayout.xml file).

To generate data from word lists, create a file with the extension ".category". The content of the file contains word or expressions - one per line. E.g. a file seasons.category would contain spring, summer autumn and winter - each on a separate line.
